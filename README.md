# usc-framework

**Unified Structural-Entropy Cosmogenesis: a Falsifiable Framework for Dark Energy, Matter Genesis and Galactic Dynamics**

The USC framework paper (consolidation of USC I/II/IV): three faces of one entropy clock, horn structure, and the falsifier matrix.

Part of the USC research program (SEDE / ECCG / APDM unification).

## Status
Pre-submission working repository, seeded 2026-07-17 from a private research hub;
full research history (including audit trails and
retractions) lives there.

## Layout
- `paper/` or `notes/` — manuscript sources
- `figures/`, `sims/`, `repro/` — figure generation and reproduction pipeline (where applicable)
- `tools/` — build tooling
